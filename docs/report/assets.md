# 资产目录

资产目录回答「平台上有哪些数据资产、谁在用、哪些该下线、哪些值得复用」。在「报表中心 → 资产目录」（`/report/assets`）管理，页面分三个标签页：**统一资产目录、可复用模板、使用与弃用**。

## 统一资产目录

跨类型汇总所有报表资源（数据源、数据集、仪表盘、指标、打印模板等），统一展示：

- 归属信息：负责人、目录、租户、生命周期状态；
- 用量计数：被查看、被引用、被**嵌入**的次数；
- 支持按类型、负责人、目录筛选，支持导出（`report:asset:export`）。

点击单个资产可查看**用量明细**（`report:asset:usage`）：访问趋势、引用它的下游资源等。

## 使用与弃用

### 用量分析

- **热门排行**：按访问量排出最常用的资产；
- **闲置资产**：列出长期无访问的资源，作为下线候选；
- **用量趋势**：观察整体访问量随时间的变化。

### 弃用公告

计划下线某个资产时，先发**弃用公告**而不是直接删除：

1. 新建公告（草稿）：选择资产、说明替代方案、设定**生效日期**；
2. 「发布」公告后进入生效日提醒，消费方可见并需主动迁移；
3. 已发布公告可**撤销发布**；到期后再实际下线资源。

## 可复用模板

把成熟的仪表盘/数据集等资源沉淀为**资产模板**，供他人一键复用：

- **新增**：从现有资源保存快照为模板；
- **克隆**：复制一份模板再修改；
- **套用（应用）**：基于模板创建新资源——应用时会**以当前用户身份重新执行权限检查**。

::: tip 安全说明
资产模板只保存允许的资源快照，**不包含数据源凭据、访问 Token 或冻结 SQL 白名单**；敏感连接信息始终由数据源的加密字段管理。
:::

## 权限

| 操作 | 权限码 |
|------|--------|
| 查看资产目录 | `report:asset:list` |
| 查看资产用量 | `report:asset:usage` |
| 导出资产目录 | `report:asset:export` |
| 弃用公告查看 / 新增 / 编辑 / 发布（含撤销） / 删除 | `report:deprecation:list` / `:create` / `:update` / `:publish` / `:delete` |
| 资产模板查看 / 新增（含克隆） / 编辑 / 套用 / 删除 | `report:asset-template:list` / `:create` / `:update` / `:apply` / `:delete` |
